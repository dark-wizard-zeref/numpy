# NumPy Notes + Tic-Tac-Toe (Streamlit)

A small learning repo containing structured NumPy practice notebooks and a simple Tic-Tac-Toe game built with Streamlit.

## Contents

- `Class_1.ipynb` — NumPy basics & Built-in methods 
- `Class_2.ipynb` — Attributes of Arrays, indexing, slicing, reshaping, Flattening 
- `Class_3.ipynb` — vectorized operations , aggregation functions, broadcasting
- `Class_4.ipynb` — Fancy Indexing, Sorting, Conditional Selection
- `tic_tac_toe.py` — Tic-Tac-Toe game using NumPy

## Requirements

- Python 3.8+
- NumPy (for core logic)
- Streamlit (for the game UI)

Install dependencies:

```bash
pip install numpy streamlit
```
